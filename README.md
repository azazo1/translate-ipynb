# Translate Ipynb Notebook

```shell
translate-ipynb --ipynb input.ipynb --output output.ipynb --lang zh --api-key sk-xxx --base-url https://aihubmix.com/v1
```

或者

```shell
export OPENAI_API_KEY=sk-xxx
export OPENAI_BASE_URL=https://aihubmix.com/v1
translate-ipynb --ipynb input.ipynb --output output.ipynb --lang zh
```
